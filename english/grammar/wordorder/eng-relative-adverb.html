関係副詞

文法と構文の説明

関係副詞は先行詞を修飾し、その文の副詞として機能します。以下の関係副詞があります：

  • where: 場所を表す（例: This is the place where I was born. - ここが私が生まれた場所です。）
  • when: 時間を表す（例: That was the day when we met. - それが私たちが出会った日です。）
  • why: 理由を表す（例: This is the reason why I left. - これが私が去った理由です。）
  • how: 方法を表す（例: I know how he solved the problem. - 彼がその問題を解決した方法を知っています。）

問題

以下の単語を並べ替えて正しい文を作成しましょう。（和訳を参考にしてください）

問題1: (the / this / where / he / lived / house / is) - これが彼が住んでいた家です。

解答: This is the house where he lived。

解説: whereは場所を表す関係副詞で、先行詞「the house」を修飾しています。

  • This: これ（代名詞）
  • is: ～です（動詞）
  • the: その（冠詞）
  • house: 家（名詞）
  • where: ～する場所（関係副詞）
  • he: 彼（代名詞）
  • lived: 住んでいた（動詞）

問題2: (day / was / when / it / the / we / met) - それが私たちが出会った日です。

解答: It was the day when we met。

解説: whenは時間を表す関係副詞で、先行詞「the day」を修飾しています。

  • It: それ（代名詞）
  • was: ～だった（動詞）
  • the: その（冠詞）
  • day: 日（名詞）
  • when: ～する時（関係副詞）
  • we: 私たち（代名詞）
  • met: 出会った（動詞）

問題3: (the / is / why / reason / this / he / angry / was) - これが彼が怒っていた理由です。

解答: This is the reason why he was angry。

解説: whyは理由を表す関係副詞で、先行詞「the reason」を修飾しています。

  • This: これ（代名詞）
  • is: ～です（動詞）
  • the: その（冠詞）
  • reason: 理由（名詞）
  • why: ～する理由（関係副詞）
  • he: 彼（代名詞）
  • was: ～だった（動詞）
  • angry: 怒った（形容詞）

問題4: (in / how / we / this / do / the / is / way / company / our / it) - これが私たちの会社でそれを行う方法です。

解答: This is the way how we do it in our company。

解説: howは「どのように」を意味する関係副詞で、「the way」を修飾しています。

  • This: これ（代名詞）
  • is: ～です（動詞）
  • the: その（冠詞）
  • way: 方法（名詞）
  • how: どのように（関係副詞）
  • we: 私たち（代名詞）
  • do: 行う（動詞）
  • it: それ（代名詞）
  • in: ～の中で（前置詞）
  • our: 私たちの（所有格）
  • company: 会社（名詞）

問題5: (will / remember / I / where / happened / it) - 私はそれがどこで起こったかを覚えています。

解答: I will remember where it happened。

解説: whereは場所を表す関係副詞で、文全体を修飾しています。

  • I: 私（代名詞）
  • will: ～するだろう（助動詞）
  • remember: 覚えている（動詞）
  • where: どこで（関係副詞）
  • it: それ（代名詞）
  • happened: 起こった（動詞）
