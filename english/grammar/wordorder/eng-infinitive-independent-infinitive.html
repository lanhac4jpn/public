不定詞 独立不定詞

文法と構文の説明

不定詞は「to + 動詞の原形」で構成され、様々な用法があります。本単元では以下を学びます：

  • in order to: ～するために
  • in order not to: ～しないために
  • so as to: ～するために
  • 独立不定詞 to tell the truth: 実を言うと

問題

以下の単語を並び替えて正しい文を作成しましょう。（和訳を参考にしてください）

問題1: (to / order / in / learn / English / I / study) - 私は英語を学ぶために勉強しています。

解答: I study in order to learn English.

解説: 「in order to + 動詞の原形」で目的を表します。

  • I: 私（代名詞）
  • study: 勉強する（動詞）
  • in order to: ～するために（句）
  • learn: 学ぶ（動詞の原形）
  • English: 英語（名詞）

問題2: (order / she / in / to / not / cry / tried) - 彼女は泣かないように努めました。

解答: She tried in order not to cry.

解説: 「in order not to」で否定の目的を表します。

  • She: 彼女（代名詞）
  • tried: 努めた（動詞）
  • in order not to: ～しないために（句）
  • cry: 泣く（動詞の原形）

問題3: (so / to / as / be / She / careful / tried) - 彼女は注意深くなるように努めました。

解答: She tried so as to be careful.

解説: 「so as to + 動詞の原形」で目的を表します。

  • She: 彼女（代名詞）
  • tried: 努めた（動詞）
  • so as to: ～するために（句）
  • be: ～である（動詞の原形）
  • careful: 注意深い（形容詞）

問題4: (tell / truth / to / the / I / tired / am) - 実を言うと、私は疲れています。

解答: To tell the truth, I am tired.

解説: 「To tell the truth」は独立不定詞で、慣用的に使用されます。

  • To tell the truth: 実を言うと（句）
  • I: 私（代名詞）
  • am: ～である（動詞）
  • tired: 疲れた（形容詞）

問題5: (it / to / started / it / make / to / matters / worse) - さらに悪いことに、雨が降り始めた。

解答: To make matters worse, it started to rain.

解説: 「To make matters worse」は独立不定詞で、状況を強調します。

  • To make matters worse: さらに悪いことに（句）
  • it: それ（代名詞）
  • started: 始めた（動詞）
  • to: ～すること（不定詞）
  • rain: 雨が降る（動詞の原形）

慣用表現の例文

  • To be sure, he is very talented. - 確かに、彼はとても才能があります。
  • So to speak, he is the backbone of our team. - いわば、彼は我々のチームの要です。
  • To tell the truth, I don't like coffee. - 実を言うと、私はコーヒーが好きではありません。
  • To begin with, we need to gather more data. - まず初めに、私たちはもっとデータを集める必要があります。
  • Strange to say, he suddenly disappeared. - 奇妙なことに、彼は突然姿を消しました。
  • To be frank with you, I think this plan has flaws. - 率直に言えば、この計画には欠陥があると思います。
  • To make matters worse, the power went out. - さらに悪いことに、停電しました。
