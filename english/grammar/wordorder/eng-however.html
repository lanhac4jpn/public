複合関係詞 However

文法と構文の説明

However は「どんなに～しても」「どんな～であっても」という意味を持つ複合関係詞です。文頭または文中で使われます。

  • 文頭にHowever: However hard you try, you cannot solve this problem. （どんなに頑張っても、この問題は解けません。）
  • 文中にHowever: You can choose, however expensive it is. （それがどんなに高価でも、選んで構いません。）

問題

以下の単語を並び替えて正しい文を作成しましょう。（和訳を参考にしてください）

問題1: (however / try / you / hard / cannot / this / solve / you / ,) - どんなに頑張っても、この問題は解けません。

解答: However hard you try, you cannot solve this problem.

解説: 文頭にHoweverを置き、程度を表現しています。

  • However: どんなに～しても（複合関係詞）
  • hard: 一生懸命（副詞）
  • you: あなた（代名詞）
  • try: 試みる（動詞）
  • cannot: ～できない（助動詞）
  • solve: 解決する（動詞）
  • this: この（指示形容詞）
  • problem: 問題（名詞）

問題2: (you / expensive / However / it / is / can / choose) - それがどんなに高価でも、選んで構いません。

解答: You can choose however expensive it is.

解説: 文中にHoweverを置き、条件を示しています。

  • you: あなた（代名詞）
  • can: ～できる（助動詞）
  • choose: 選ぶ（動詞）
  • however: どんなに～しても（複合関係詞）
  • expensive: 高価な（形容詞）
  • it: それ（代名詞）
  • is: ～である（動詞）

問題3: (However / fast / you / run / will / not / catch / him / you / ,) - どんなに速く走っても、彼を捕まえられません。

解答: However fast you run, you will not catch him.

解説: 文頭にHoweverを置いて速度を表現しています。

  • However: どんなに～しても（複合関係詞）
  • fast: 速く（副詞）
  • you: あなた（代名詞）
  • run: 走る（動詞）
  • will: ～だろう（助動詞）
  • not: ～ない（副詞）
  • catch: 捕まえる（動詞）
  • him: 彼を（代名詞）

問題4: (to solve / this problem / you / approach it / hard / however / is / ,) - どんなふうに取り組んでも、この問題を解くのは難しい。

解答: However you approach it, this problem is hard to solve.

解説: 文頭でHoweverを補足的に使用しています。

  • however: どんなに～しても（複合関係詞）
  • you: あなた（代名詞）
  • approach: 取り組む（動詞）
  • it: それ（代名詞）
  • this: この（指示形容詞）
  • problem: 問題（名詞）
  • is: ～である（動詞）
  • hard: 難しい（形容詞）
  • to: ～すること（不定詞）
  • solve: 解決する（動詞）

問題5: (However / you / try / fail / will / efforts / your) - どんなに試みても、努力は失敗に終わるでしょう。

解答: However you try, your efforts will fail.

解説: 文頭にHoweverを置いて試行を示しています。

  • However: どんなに～しても（複合関係詞）
  • you: あなた（代名詞）
  • try: 試みる（動詞）
  • your: あなたの（所有格）
  • efforts: 努力（名詞）
  • will: ～だろう（助動詞）
  • fail: 失敗する（動詞）
