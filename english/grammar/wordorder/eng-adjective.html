形容詞

文法と構文の説明

形容詞は名詞を修飾して、その性質や状態、数量などを説明する言葉です。以下はよく使われる形容詞の例です。

  • big（大きい）
  • small（小さい）
  • happy（幸せな）
  • sad（悲しい）
  • interesting（面白い）
  • important（重要な）
  • beautiful（美しい）
  • fast（速い）
  • slow（遅い）
  • clean（きれいな）

問題

以下の単語を並び替えて正しい文を作成しましょう。（和訳を参考にしてください）

問題1: (is / this / big / a / house) - これは大きな家です。

解答: This is a big house.

解説: 形容詞「big」は名詞「house」を修飾します。

  • This: これは（代名詞）
  • is: ～です（動詞）
  • a: 1つの（冠詞）
  • big: 大きい（形容詞）
  • house: 家（名詞）

問題2: (happy / looks / he / very) - 彼はとても幸せそうです。

解答: He looks very happy.

解説: 「looks」は感覚動詞で、形容詞「happy」を続けます。

  • He: 彼（代名詞）
  • looks: ～に見える（動詞）
  • very: とても（副詞）
  • happy: 幸せな（形容詞）

問題3: (is / the / beautiful / flower) - その花は美しいです。

解答: The flower is beautiful.

解説: 「beautiful」は名詞「flower」を説明します。

  • The: その（冠詞）
  • flower: 花（名詞）
  • is: ～です（動詞）
  • beautiful: 美しい（形容詞）

問題4: (clean / is / this / room) - この部屋はきれいですです。

解答: This room is cleanh.

解説: 「clean」は「room」を説明します。

  • This: この（代名詞）
  • room: 部屋（名詞）
  • is: ～です（動詞）
  • clean: きれいな（形容詞）

問題5: (fast / the / car / is) - その車は速いです。

解答: The car is fast.

解説: 「fast」は名詞「car」を修飾します。

  • The: その（冠詞）
  • car: 車（名詞）
  • is: ～です（動詞）
  • fast: 速い（形容詞）
