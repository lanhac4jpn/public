疑問詞 How

文法と構文の説明

How は「どのように」「どれくらい」などを尋ねる疑問詞です。以下のようにさまざまな形式で使用されます：

  • How + be動詞 - 状態や調子を尋ねる。
  • How + Do + S + V - 動作や方法について尋ねる。

例文：

  • How is your family? （ご家族の調子はどうですか？）
  • How do you go to school? （どうやって学校に行きますか？）

問題

以下の単語を並び替えて正しい文を作成しましょう。（和訳を参考にしてください）

問題1: (the weather / how / today / is / ?) - 今日の天気はどうですか？

解答: How is the weather today?

解説: 「How + be動詞」を用いて状態を尋ねます。

  • How: どのように（疑問詞）
  • is: ～である（be動詞）
  • the: その（冠詞）
  • weather: 天気（名詞）
  • today: 今日（副詞）

問題2: ( do / how / study / English / you / ?) - あなたはどのように英語を勉強しますか？

解答: How do you study English?

解説: 「How + Do + S + V」を用いて動作について尋ねます。

  • How: どのように（疑問詞）
  • do: ～する（助動詞）
  • you: あなた（代名詞）
  • study: 勉強する（動詞）
  • English: 英語（名詞）

問題3: (your / new job / is / going / how / ?) - 新しい仕事はどうですか？

解答: How is your new job going?

解説: 「How + be動詞」を用いて進行状況を尋ねます。

  • How: どのように（疑問詞）
  • is: ～である（be動詞）
  • your: あなたの（所有格）
  • new: 新しい（形容詞）
  • job: 仕事（名詞）
  • going: 進んでいる（現在進行形）

問題4: (solve / do / they / how / this problem / ?) - 彼らはどうやってこの問題を解決しますか？

解答: How do they solve this problem?

解説: 「How + Do + S + V」を用いて方法を尋ねます。

  • How: どのように（疑問詞）
  • do: ～する（助動詞）
  • they: 彼ら（代名詞）
  • solve: 解決する（動詞）
  • this: この（指示形容詞）
  • problem: 問題（名詞）

問題5: (recently / is / your health / how / ?) - 最近、体調はどうですか？

解答: How is your health recently?

解説: 「How + be動詞」を用いて体調を尋ねます。

  • How: どのように（疑問詞）
  • is: ～である（be動詞）
  • your: あなたの（所有格）
  • health: 健康（名詞）
  • recently: 最近（副詞）

補足例文

  • How many books do you have? （本を何冊持っていますか？）
  • How much is this? （これはいくらですか？）
  • How long does it take? （どのくらい時間がかかりますか？）
  • How often do you go there? （どのくらいの頻度でそこに行きますか？）
  • How about meeting tomorrow? （明日会うのはどうですか？）
  • How far is the station? （駅はどのくらい遠いですか？）
  • How old are you? （何歳ですか？）
