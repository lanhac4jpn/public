go 「行く」

文法説明

「go」は「行く」という意味の動詞です。主語や時制によって形が変わります。

  • 現在形: I, You, We, They → go
  • 三人称単数: He, She, It → goes
  • 未来形: will go

問題

以下の単語を正しい語順に並べ替えて、正しい英文を作りましょう。

問題 1: (school / I / to / go / day / every) - 私は毎日学校に行きます。

解答: I go to school every day.

解説: 「I go to school every day.」は「私は毎日学校に行きます。」という意味です。

  • I: 私（代名詞）
  • go: 行く（動詞）
  • to: 〜へ（前置詞）
  • school: 学校（名詞）
  • every: 毎〜（形容詞）
  • day: 日（名詞）

問題 2: (to / goes / the / park / she) - 彼女は公園に行きます。

解答: She goes to the park.

解説: 「She goes to the park.」は「彼女は公園に行きます。」という意味です。

  • She: 彼女（代名詞）
  • goes: 行く（三単現）（動詞）
  • to: 〜へ（前置詞）
  • the: その（冠詞）
  • park: 公園（名詞）

問題 3: (will / we / shopping / go / tomorrow) - 私たちは明日買い物に行きます。

解答: We will go shopping tomorrow.

解説: 「We will go shopping tomorrow.」は「私たちは明日買い物に行きます。」という意味です。

  • We: 私たち（代名詞）
  • will: 〜するつもり（助動詞）
  • go: 行く（動詞）
  • shopping: 買い物（動名詞）
  • tomorrow: 明日（副詞）

  • the: その（冠詞）

問題 5: (goes / to / he / at / bed / 9 / o’clock) - 彼は9時に寝ます。

解答: He goes to bed at 9 o’clock.

解説: 「He goes to bed at 9 o’clock.」は「彼は9時に寝ます。」という意味です。

  • He: 彼（代名詞）
  • goes: 行く（三単現）（動詞）
  • to: 〜へ（前置詞）
  • bed: ベッド（寝る）（名詞）
  • at: 〜に（前置詞）
  • 9: 9（数詞）
  • o’clock: 時（名詞）

問題 6: (go / Let's / a /for / walk) - 散歩に行きましょう。

解答: Let's go for a walk.

解説: 「Let’s go for a walk.」は「散歩に行きましょう。」という意味です。

  • Let’s: 〜しましょう（助動詞+動詞）
  • go: 行く（動詞）
  • for: 〜のために（前置詞）
  • a: 1つの（冠詞）
  • walk: 散歩（名詞）

問題 7: (to / want / go / I / zoo / to / the) - 私は動物園に行きたいです。

解答: I want to go to the zoo.

解説: 「I want to go to the zoo.」は「私は動物園に行きたいです。」という意味です。

  • I: 私（代名詞）
  • want: 〜したい（動詞）
  • to: 〜すること（不定詞）
  • go: 行く（動詞）
  • to: 〜へ（前置詞）
  • the: その（冠詞）
  • zoo: 動物園（名詞）

  • The: その（冠詞）

問題 9: (too / go / don't / far) - 遠くに行きすぎないでね！

解答: Don't go too far!

解説: 「Don’t go too far!」は「遠くに行きすぎないでね！」という意味です。

  • Don’t: 〜してはいけない（助動詞+否定）
  • go: 行く（動詞）
  • too: あまりに（副詞）
  • far: 遠くに（副詞）

問題 10: (we / theater / go / the / to / can / movie / ?) - 映画館に行ってもいいですか？

解説: 「Can we go to the movie theater?」は「映画館に行ってもいいですか？」という意味です。

  • Can: 〜してもよいですか（助動詞）
  • we: 私たち（代名詞）
  • go: 行く（動詞）
  • to: 〜へ（前置詞）
  • the: その（冠詞）
  • movie: 映画（名詞）
  • theater: 劇場（名詞）
