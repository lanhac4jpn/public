現在完了形 経験

文法と構文の説明

現在完了形 経験は、「これまでに～したことがある」という経験を表します。

現在完了形の構造は次の通りです：

  • 肯定文: 主語 + have/has + 過去分詞 + ...
  • 疑問文: Have/Has + 主語 + 過去分詞 + ...?
  • 否定文: 主語 + have/has + not + 過去分詞 + ...

例文：

  • I have visited Kyoto.（私は京都を訪れたことがあります。）
  • Have you ever visited Kyoto?（あなたは京都を訪れたことがありますか？）
  • I have not visited Kyoto.（私は京都を訪れたことがありません。）

問題

以下の単語を並び替えて正しい文を作成しましょう。（和訳を参考にしてください）

問題1: (Kyoto / visited / have / I) - 私は京都を訪れたことがあります。

解答: I have visited Kyoto。

解説: 現在完了形の肯定文です。「have + 過去分詞」を使います。

  • I: 私（主語）
  • have: ～したことがある（助動詞）
  • visited: 訪れた（過去分詞）
  • Kyoto: 京都（名詞）

問題2: (you / Have / visited / ever / Kyoto / ?) - あなたは京都を訪れたことがありますか？

解答: Have you ever visited Kyoto?

解説: 現在完了形の疑問文です。「Have + 主語 + 過去分詞」の形を使います。

  • Have: ～したことがありますか？（助動詞）
  • you: あなた（主語）
  • ever: これまでに（副詞）
  • visited: 訪れた（過去分詞）
  • Kyoto: 京都（名詞）

問題3: (not / have / visited / I / Kyoto) - 私は京都を訪れたことがありません。

解答: I have not visited Kyoto。

解説: 現在完了形の否定文です。「have not + 過去分詞」を使います。

  • I: 私（主語）
  • have: ～したことがある（助動詞）
  • not: ～ない（否定語）
  • visited: 訪れた（過去分詞）
  • Kyoto: 京都（名詞）

問題4: (seen / have / never / I / Mount Fuji) - 私は富士山を一度も見たことがありません。

解答: I have never seen Mount Fuji。

解説: 「never」は「一度も～ない」という否定の意味を持ちます。

  • I: 私（主語）
  • have: ～したことがある（助動詞）
  • never: 一度も～ない（副詞）
  • seen: 見た（過去分詞）
  • Mount Fuji: 富士山（名詞）

問題5: (been / ever / have / to / you / Paris / ?) - あなたはかつてパリに行ったことがありますか？

解答: Have you ever been to Paris?

解説: 「been to」は「～に行ったことがある」という意味を表します。

  • Have: ～したことがありますか？（助動詞）
  • you: あなた（主語）
  • ever: かつて（副詞）
  • been: 行ったことがある（過去分詞）
  • to: ～へ（前置詞）
  • Paris: パリ（名詞）
