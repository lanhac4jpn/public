be動詞 なる

問題

以下の単語を正しい語順に並べ替えて、正しい英文を作りましょう。

問題 1: (be / I / want / a / teacher / to / .) - 私は先生になりたいです。

解答: I want to be a teacher.

問題 2: (became / doctor / he / a / .) - 彼は医者になりました。

解答: He became a doctor.

問題 3: (is / warm / the / getting / weather / .) - 天気が暖かくなっています。

解答: The weather is getting warm.

問題 4: (was / big / the / baby / becoming / .) - その赤ちゃんは大きくなっていました。

解答: The baby was becoming big.

問題 5: (be / want / she / singer / to / a / .) - 彼女は歌手になりたいです。

解答: She wants to be a singer.

問題 6: (be / famous / will / one / he / day / .) - 彼はいつか有名になるでしょう。

解答: He will be famous one day.

問題 7: (became / quiet / room / the / .) - 部屋が静かになりました。

解答: The room became quiet.

問題 8: (becomes / red / the / sky / evening / in / .) - 夕方になると空が赤くなります。

解答: The sky becomes red in the evening.

問題 9: (be / happy / want / we / to / .) - 私たちは幸せになりたいです。

解答: We want to be happy.
