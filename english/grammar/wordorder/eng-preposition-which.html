関係代名詞 前置詞 + Which

文法説明

「前置詞 + which」は、関係代名詞を使って前置詞と関係詞節を結びつける構文です。この形を用いることで、より形式的で正確な表現を作ることができます。

  • 例: This is the book about which I told you. （これが私が君に話した本です。）
  • 例: The house in which he lives is old. （彼が住んでいる家は古いです。）

問題

以下の単語を並べ替えて正しい文章を作りましょう。（和訳も参考にしてください）

問題1: (which / I / you / is / told / the / book / about / this) - これが私が君に話した本です。

解答: This is the book about which I told you。

解説: 「about which」を使うことで、文を形式的で正確な表現にしています。

  • This: これ（代名詞）
  • is: ～です（be動詞）
  • the: その（冠詞）
  • book: 本（名詞）
  • about: ～について（前置詞）
  • which: ～するところの（関係代名詞）
  • I: 私は（代名詞、主格）
  • told: 話した（動詞、過去形）
  • you: 君に（代名詞、目的格）

問題2: (he / in / the / house / which / is / lives / old) - 彼が住んでいる家は古いです。

解答: The house in which he lives is old。

解説: 「in which」を使うことで、住む場所を表しています。

  • The: その（冠詞）
  • house: 家（名詞）
  • in: ～の中に（前置詞）
  • which: ～するところの（関係代名詞）
  • he: 彼は（代名詞、主格）
  • lives: 住む（動詞、三単現）
  • is: ～です（be動詞）
  • old: 古い（形容詞）

問題3: (goes / to / which / the / is / school / famous / she) - 彼女が通っている学校は有名です。

解答: The school to which she goes is famous。

解説: 「to which」を使って、目的地を表します。

  • The: その（冠詞）
  • school: 学校（名詞）
  • to: ～へ（前置詞）
  • which: ～するところの（関係代名詞）
  • she: 彼女は（代名詞、主格）
  • goes: 行く（動詞、三単現）
  • is: ～です（be動詞）
  • famous: 有名な（形容詞）

問題4: (was / chair / he / on / the / sat / which / broken) - 彼が座った椅子は壊れていました。

解答: The chair on which he sat was broken。

解説: 「on which」を使って、物理的な接触を表します。

  • The: その（冠詞）
  • chair: 椅子（名詞）
  • on: ～の上に（前置詞）
  • which: ～するところの（関係代名詞）
  • he: 彼は（代名詞、主格）
  • sat: 座った（動詞、過去形）
  • was: ～でした（be動詞、過去形）
  • broken: 壊れた（形容詞）

問題5: (we / for / which / successful / the / worked / project / was) - 私たちが取り組んだプロジェクトは成功しました。

解答: The project for which we worked was successful。

解説: 「for which」を使って、目的や理由を表します。

  • The: その（冠詞）
  • project: プロジェクト（名詞）
  • for: ～のために（前置詞）
  • which: ～するところの（関係代名詞）
  • we: 私たちは（代名詞、主格）
  • worked: 働いた（動詞、過去形）
  • was: ～でした（be動詞、過去形）
  • successful: 成功した（形容詞）
