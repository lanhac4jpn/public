一般動詞 過去形・不規則動詞

文法と構文の説明

不規則動詞は過去形を作る際に規則に従わない変化をします。これらの形は覚える必要があります。

  • 肯定文: 主語 + 動詞の過去形
    例: I went to school. （私は学校へ行きました）
  • 疑問文: Did + 主語 + 動詞の原形?
    例: Did you go to school? （あなたは学校へ行きましたか？）
  • 否定文: 主語 + did not + 動詞の原形
    例: I did not go to school. （私は学校へ行きませんでした）

問題

以下の単語を並べ替えて正しい文を作成しましょう。（和訳を参考にしてください）

問題1: (went / to / park / the / he / yesterday) - 彼は昨日、公園に行きました。

解答: He went to the park yesterday。

解説: 不規則動詞「go」の過去形は「went」です。

  • He: 彼（代名詞）
  • went: 行った（動詞 過去形）
  • to: ～へ（前置詞）
  • the: その（冠詞）
  • park: 公園（名詞）
  • yesterday: 昨日（副詞）

問題2: (ate / she / an / apple / morning / this) - 彼女は今朝、リンゴを食べました。

解答: She ate an apple this morning。

解説: 不規則動詞「eat」の過去形は「ate」です。

  • She: 彼女（代名詞）
  • ate: 食べた（動詞 過去形）
  • an: 1つの（冠詞）
  • apple: リンゴ（名詞）
  • this morning: 今朝（副詞句）

問題3: (you / letter / write / a / did / last week / ?) - あなたは先週、手紙を書きましたか？

解答: Did you write a letter last week?

解説: 疑問文では「Did」を使い、動詞「write」は原形を用います。

  • Did: ～しましたか（助動詞）
  • you: あなた（代名詞）
  • write: 書く（動詞 原形）
  • a: 1つの（冠詞）
  • letter: 手紙（名詞）
  • last week: 先週（副詞句）

問題4: (not / see / we / yesterday / did / the / movie ) - 私たちは昨日、その映画を見ませんでした。

解答: We did not see the movie yesterday。

解説: 否定文では「did not」を使い、動詞「see」は原形を用います。

  • We: 私たち（代名詞）
  • did not: ～しませんでした（助動詞 + 否定形）
  • see: 見る（動詞 原形）
  • the: その（冠詞）
  • movie: 映画（名詞）
  • yesterday: 昨日（副詞）

問題5: (made / a / last night / cake / they) - 彼らは昨夜、ケーキを作りました。

解答: They made a cake last night。

解説: 不規則動詞「make」の過去形は「made」です。

  • They: 彼ら（代名詞）
  • made: 作った（動詞 過去形）
  • a: 1つの（冠詞）
  • cake: ケーキ（名詞）
  • last night: 昨夜（副詞句）
