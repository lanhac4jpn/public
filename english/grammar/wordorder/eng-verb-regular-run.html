run 「走る・流れる・動作する」

文法説明

「run」は「速く動く」「流れる」「機械などが動く」「経営する」など、様々な意味を持つ動詞です。

  • 現在形： run / runs（三単現）
  • 過去形： ran
  • 現在分詞： running
  • 過去分詞： run

問題

以下の単語を正しい語順に並べ替えて、正しい英文を作りましょう。

問題 1： (run / fast / I / can) - 私は速く走ることができます。

解答： I can run fast.

解説： 「I can run fast.」は「私は速く走ることができます。」という意味です。

  • I： 私（代名詞）
  • can： 〜できる（助動詞）
  • run： 走る（動詞）
  • fast： 速く（副詞）

問題 2： (runs / every / she / morning) - 彼女は毎朝走ります。

解答： She runs every morning.

解説： 「She runs every morning.」は「彼女は毎朝走ります。」という意味です。

  • She： 彼女（代名詞）
  • runs： 走る（三単現）（動詞）
  • every： 毎（形容詞）
  • morning： 朝（名詞）

問題 3： (to / he / ran / the / park) - 彼は公園まで走りました。

  • He： 彼（代名詞）
  • ran： 走った（動詞（過去形））
  • to： 〜へ（前置詞）
  • the： その（冠詞）
  • park： 公園（名詞）
