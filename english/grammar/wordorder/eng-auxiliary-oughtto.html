助動詞表現 ought to

文法と構文の説明

助動詞表現 ought to は、「～するべきだ」「～したほうがよい」という意味を表します。

  • 肯定文: 主語 + ought to + 動詞の原形
  • 疑問文: Ought + 主語 + to + 動詞の原形?
  • 否定文: 主語 + ought not to + 動詞の原形

例文：

  • 肯定文: You ought to see a doctor. （あなたは医者に診てもらうべきです。）
  • 疑問文: Ought I to apologize? （私は謝るべきですか？）
  • 否定文: He ought not to waste money. （彼はお金を無駄にすべきではありません。）

問題

以下の単語を並び替えて正しい文を作成しましょう。（和訳を参考にしてください）

問題1: (ought / to / you / call / him) - あなたは彼に電話するべきです。

解答: You ought to call him.

解説: 肯定文では「主語 + ought to + 動詞の原形」を使います。

  • You: あなた（主語）
  • ought to: ～すべきだ（助動詞表現）
  • call: 電話する（動詞の原形）
  • him: 彼に（目的語）

問題2: (I / apologize / to / ought / ?) - 私は謝るべきですか？

解答: Ought I to apologize?

解説: 疑問文では「Ought + 主語 + to + 動詞の原形」を使います。

  • Ought: ～すべきですか？（助動詞表現）
  • I: 私（主語）
  • to: ～へ（助詞）
  • apologize: 謝る（動詞の原形）

問題3: (ought / waste / not / he / to / time) - 彼は時間を無駄にすべきではありません。

解答: He ought not to waste time.

解説: 否定文では「主語 + ought not to + 動詞の原形」を使います。

  • He: 彼（主語）
  • ought not to: ～すべきではない（助動詞表現の否定形）
  • waste: 無駄にする（動詞の原形）
  • time: 時間（名詞）

問題4: (ought / to / they / help / each other) - 彼らはお互いに助け合うべきです。

解答: They ought to help each other.

解説: 肯定文の形式です。

  • They: 彼ら（主語）
  • ought to: ～すべきだ（助動詞表現）
  • help: 助ける（動詞の原形）
  • each other: お互いに（副詞句）

問題5: (ought / not / we / to / argue) - 私たちは議論すべきではありません。

解答: We ought not to argue.

解説: 否定文で「ought not to」を使います。

  • We: 私たち（主語）
  • ought not to: ～すべきではない（助動詞表現の否定形）
  • argue: 議論する（動詞の原形）
