受動態

文法と構文の説明

受動態（Passive Voice）は、動作の主体ではなく動作を受ける側を強調する際に用いられる構文です。

基本形: be動詞 + 過去分詞

例:

  • 肯定文: The book was written by him. (本は彼によって書かれました。)
  • 疑問文: Was the book written by him? (本は彼によって書かれましたか？)
  • 否定文: The book wasn't written by him? (本は彼によって書かれませんでした。)

問題

以下の単語を並び替えて正しい文を作成しましょう。（和訳を参考にしてください）

問題1: (written / the book / was / by him) - その本は彼によって書かれた。

解答: The book was written by him.

解説: 受動態では「be動詞 + 過去分詞」が用いられ、動作の主体を示す場合は「by + 動作主」を付加します。

  • the book: その本（名詞）
  • was: ～であった（be動詞）
  • written: 書かれた（過去分詞）
  • by him: 彼によって（前置詞句）

問題2: (is / by many people / spoken / English) - 英語は多くの人に話されている。

解答: English is spoken by many people.

解説: 現在形の受動態では「is + 過去分詞」を使います。

  • English: 英語（名詞）
  • is: ～である（be動詞）
  • spoken: 話されている（過去分詞）
  • by many people: 多くの人によって（前置詞句）

問題3: (was / not / the room / cleaned / by her) - その部屋は彼女によって掃除されなかった。

解答: The room was not cleaned by her.

解説: 否定文では「not」を「be動詞」の後に配置します。

  • the room: その部屋（名詞）
  • was not: ～でなかった（be動詞 + 否定語）
  • cleaned: 掃除された（過去分詞）
  • by her: 彼女によって（前置詞句）

問題4: (painted / who / was / this picture / ?) - この絵は誰が描いたのですか？

解答: Who was this picture painted?

解説: 疑問文では「by + 疑問詞」が文頭に来ます。

  • Who: 誰が（疑問詞句）
  • was: ～であった（be動詞）
  • this picture: この絵（名詞句）
  • painted: 描かれた（過去分詞）

問題5: (are / they / not / invited / by him / ?) - 彼らは彼によって招待されていないのですか？

解答: Are they not invited by him?

解説: 疑問文の否定形では「are + 主語 + not + 過去分詞」の語順を用います。

  • are: ～ですか（be動詞）
  • they: 彼ら（代名詞）
  • not: ～でない（否定語）
  • invited: 招待された（過去分詞）
  • by him: 彼によって（前置詞句）
