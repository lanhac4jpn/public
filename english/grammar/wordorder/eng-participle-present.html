現在分詞の後置修飾（主語と動詞を含む）

文法説明

現在分詞が名詞を後ろから修飾するとき、主語と動詞（現在分詞）を含む句の形を取ることがあります。

和訳: 名詞を説明する文のような情報（誰が何をしているか）を、現在分詞で短くして名詞の後に置く使い方です。

  • a girl sleeping on the bed：ベッドで寝ている女の子（who is sleeping... の短縮）
  • a boy drawing a picture：絵を描いている男の子（who is drawing... の短縮）

問題

次の語を正しい語順に並べ替えて、意味のある文にしてください。

問題1: (a / girl / there / on / the / sleeping / bed / is) - ベッドで寝ている女の子がいます。

解答: There is a girl sleeping on the bed.

解説: 「a girl sleeping on the bed」で「ベッドで寝ている女の子」。この部分は who is sleeping... の意味を省略して現在分詞を用いた修飾です。

  • there: そこに（副詞）
  • is: ～がある（動詞）
  • a: 一人の（冠詞）
  • girl: 女の子（名詞）
  • sleeping: 眠っている（現在分詞）
  • on: ～の上に（前置詞）
  • the: その（冠詞）
  • bed: ベッド（名詞）

問題2: (here / boy / a / a / drawing / picture / is) - 絵を描いている男の子がここにいます。

解答: A boy drawing a picture is here.

解説: 「drawing a picture」が「絵を描いている」で boy を後ろから修飾します。

  • a: 一人の（冠詞）
  • boy: 男の子（名詞）
  • drawing: 描いている（現在分詞）
  • a: 一つの（冠詞）
  • picture: 絵（名詞）
  • is: ～です（動詞）
  • here: ここに（副詞）

問題3: (children / running / the / in / park / happy / are) - 公園で走っている子供たちは幸せです。

解答: The children running in the park are happy.

解説: 「running in the park」で「公園で走っている」を表し、「children」を修飾します。

  • the: その（冠詞）
  • children: 子供たち（名詞）
  • running: 走っている（現在分詞）
  • in: ～の中に（前置詞）
  • the: その（冠詞）
  • park: 公園（名詞）
  • are: ～です（動詞）
  • happy: 幸せな（形容詞）

問題4: (teacher / a / sitting / the / on / wating / chair / is) - 椅子に座っている先生は待っています。

解答: A teacher sitting on the chair is waiting.

解説: 「sitting on the chair」で「椅子に座っている」を teacher にかけています。

  • a: 一人の（冠詞）
  • teacher: 先生（名詞）
  • sitting: 座っている（現在分詞）
  • on: ～の上に（前置詞）
  • the: その（冠詞）
  • chair: 椅子（名詞）
  • is: ～している（動詞）
  • waiting: 待っている（動詞）

問題5: (my / the / father / man / is / standing / door / near / the) - ドアの近くに立っている男の人は私の父です。

解答: The man standing near the door is my father.

解説: 「standing near the door」が「ドアの近くに立っている」で man を後ろから説明します。

  • the: その（冠詞）
  • man: 男性（名詞）
  • standing: 立っている（現在分詞）
  • near: ～の近くに（前置詞）
  • the: その（冠詞）
  • door: ドア（名詞）
  • is: ～です（動詞）
  • my: 私の（代名詞）
  • father: 父（名詞）
