There is / There are 構文

文法説明

「There is / There are」は、「～があります」「～がいます」という存在を表す構文です。

後ろに来る名詞が単数なら There is、複数なら There are を使います。

例:

  • 単数: There is a book on the desk.（机の上に本があります。）
  • 複数: There are two cats in the room.（部屋に猫が2匹います。）
  • 疑問文: Is there a park near here?（この近くに公園はありますか？）
  • 否定文: There is not a library in this town.（この町には図書館がありません。）

用語解説

  • There is: 単数のもの・人が「ある」「いる」ときに使う表現。
  • There are: 複数のもの・人が「ある」「いる」ときに使う表現。
  • 単数名詞: 1つ・1人を表す名詞。
  • 複数名詞: 2つ以上・2人以上を表す名詞。

短縮形

  • There is は There’s と短縮できます。
  • There are は、一般的な書き言葉ではあまり短縮しません。
  • There is not は There isn’t と短縮できます。
  • There are not は There aren’t と短縮できます。

問題

以下の単語を並べ替えて正しい文章を作りましょう。（和訳も参考にしてください）

問題1: (is / a book / there / on the desk) - 机の上に本があります。

解答: There is a book on the desk.

解説: 「a book」は単数なので、There is を使います。

  • There: そこに・存在を表す語（副詞）
  • is: ～がある・いる（動詞）
  • a book: 1冊の本（名詞句）
  • on the desk: 机の上に（前置詞句）

問題2: (are / two cats / there / in the room) - 部屋に猫が2匹います。

解答: There are two cats in the room.

解説: 「two cats」は複数なので、There are を使います。

  • There: そこに・存在を表す語（副詞）
  • are: ～がある・いる（動詞）
  • two cats: 2匹の猫（名詞句）
  • in the room: 部屋の中に（前置詞句）

問題3: (is / there / a park / near here / ?) - この近くに公園はありますか？

解答: Is there a park near here?

解説: 疑問文では、be動詞 is を文頭に置きます。

  • Is: ～がありますか・いますか（動詞）
  • there: 存在を表す語（副詞）
  • a park: 1つの公園（名詞句）
  • near here: この近くに（副詞句）

問題4: (are / many students / there / in the classroom) - 教室にたくさんの生徒がいます。

解答: There are many students in the classroom.

解説: 「many students」は複数なので、There are を使います。

  • There: そこに・存在を表す語（副詞）
  • are: ～がある・いる（動詞）
  • many students: たくさんの生徒（名詞句）
  • in the classroom: 教室の中に（前置詞句）

問題5: (is not / a library / there / in this town) - この町には図書館がありません。

解答: There is not a library in this town.

解説: 否定文では、be動詞の後ろに not を置きます。短縮形では There isn’t a library in this town. と言えます。

  • There: そこに・存在を表す語（副詞）
  • is not: ～がない・いない（動詞＋否定語）
  • a library: 1つの図書館（名詞句）
  • in this town: この町に（前置詞句）
