want 「欲しい・〜したい」

文法説明

「want」は「〜が欲しい」「〜したい」といった希望や願望を表す動詞です。後ろに名詞や「to + 動詞」を続ける形で使われます。

  • 現在形： want / wants（三単現）
  • 過去形： wanted
  • 否定形： don’t want / doesn’t want

問題

以下の単語を正しい語順に並べ替えて、正しい英文を作りましょう。

問題 1： (apple / want / I / an) - 私はりんごが欲しいです。

解答： I want an apple.

解説： 「I want an apple.」は「私はりんごが欲しいです。」という意味です。

  • I： 私（代名詞）
  • want： 欲しい（動詞）
  • an： 1つの（冠詞）
  • apple： りんご（名詞）

問題 2： (you / do / want / water / some / ?) - あなたは水が欲しいですか？

解答： Do you want some water?

解説： 「Do you want some water?」は「あなたは水が欲しいですか？」という意味です。

  • Do： 〜ですか（助動詞）
  • you： あなた（代名詞）
  • want： 欲しい（動詞）
  • some： いくらかの（形容詞）
  • water： 水（名詞）

問題 3： (wants / a / she / new / bag) - 彼女は新しいかばんが欲しいです。
