助動詞 Should

文法と構文の説明

助動詞 Should は、「～すべき」「～したほうがよい」という義務や助言を表します。 また、推測の意味で「～するはずだ」と使われることもあります。

  • 肯定文: 主語 + should + 動詞の原形
  • 疑問文: Should + 主語 + 動詞の原形?
  • 否定文: 主語 + should not (shouldn't) + 動詞の原形

例文：

  • 肯定文: You should study more. （もっと勉強するべきです。）
  • 疑問文: Should I apologize to him? （彼に謝るべきでしょうか？）
  • 否定文: You shouldn't eat too much. （食べ過ぎるべきではありません。）

問題

以下の単語を並び替えて正しい文を作成しましょう。（和訳を参考にしてください）

問題1: (a taxi / we / take / should / to the station / ?) - 私たちは駅までタクシーを利用するべきですか？

解答: Should we take a taxi to the station?

解説: 疑問文では「Should」を主語の前に置きます。

  • Should: ～すべきか（助動詞）
  • we: 私たち（主語）
  • take: 取る、乗る（動詞の原形）
  • a taxi: タクシー（名詞）
  • to the station: 駅まで（前置詞句）

問題2: (your homework / finish / should / you / before dinner) - あなたは夕食前に宿題を終わらせるべきです。

解答: You should finish your homework before dinner.

解説: 肯定文では「主語 + should + 動詞の原形」を使います。

  • You: あなた（主語）
  • should: ～すべき（助動詞）
  • finish: 終わらせる（動詞の原形）
  • your homework: あなたの宿題（名詞句）
  • before dinner: 夕食前に（前置詞句）

問題3: (stay up / should not / so late / you) - あなたはそんなに遅くまで起きているべきではありません。

解答: You should not stay up so late.

解説: 否定文では「should not（shouldn't）」を使い、助言を否定します。

  • You: あなた（主語）
  • should not: ～すべきではない（助動詞の否定形）
  • stay up: 起きている（動詞句）
  • so late: そんなに遅く（副詞句）

問題4: (my parents / I / call / should / now / ?) - 私は今、両親に電話をするべきですか？

解答: Should I call my parents now?

解説: 疑問文では「Should」を主語の前に置きます。

  • Should: ～すべきか（助動詞）
  • I: 私（主語）
  • call: 電話をする（動詞の原形）
  • my parents: 私の両親（名詞句）
  • now: 今（副詞）

問題5: (should / you / regularly / exercise ) - あなたは定期的に運動するべきです。

解答: You should exercise regularly.

解説: 助言として「should」を使います。

  • You: あなた（主語）
  • should: ～すべき（助動詞）
  • exercise: 運動する（動詞의 原形）
  • regularly: 定期的に（副詞）
