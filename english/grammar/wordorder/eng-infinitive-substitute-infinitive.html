不定詞 代不定詞

文法と構文の説明

不定詞は「to + 動詞の原形」の形を取り、名詞的、形容詞的、副詞的な働きをします。以下のような形で使用されます：

  • not to be: ～しないこと
  • 代不定詞 not to: 前述の内容を繰り返さずに否定の意味を表現する
  • 進行形 to be Ving: ～していること
  • 受動態 to be 過去分詞: ～されること
  • 完了不定詞 to have 過去分詞: ～したこと

問題

以下の単語を並べ替えて正しい文を作成しましょう。（和訳を参考にしてください）

問題1: (to / be / want / not / they / there) - 彼らはそこにいたくない。

解答: They want not to be there.

解説: 「not to be」を使って否定の意味を表現しています。

  • They: 彼ら（代名詞）
  • want: ～したい（動詞）
  • not: ～しない（副詞）
  • to: ～すること（不定詞）
  • be: いる（動詞の原形）
  • there: そこ（副詞）

問題2: (I / not / promised / to / do / that) - 私はそれをしないと約束しました。

解答: I promised not to do that.

解説: 「not to」を使って否定の意味を表現しています。

  • I: 私（代名詞）
  • promised: 約束した（動詞）
  • not: ～しない（副詞）
  • to: ～すること（不定詞）
  • do: する（動詞の原形）
  • that: それ（代名詞）

問題3: (to / studying / be / seems / She) - 彼女は勉強しているように見える。

解答: She seems to be studying.

解説: 「to be Ving」を使って進行形の不定詞を表現しています。

  • She: 彼女（代名詞）
  • seems: ～のようだ（動詞）
  • to: ～すること（不定詞）
  • be: ～である（動詞の原形）
  • studying: 勉強している（動名詞）

問題4: (be / to / found / wants / The book) - その本は見つけられたいと思っています。

解答: The book wants to be found.

解説: 「to be 過去分詞」を使って受動態の不定詞を表現しています。

  • The book: その本（名詞句）
  • wants: ～したい（動詞）
  • to: ～すること（不定詞）
  • be: ～される（動詞の原形）
  • found: 見つけられる（過去分詞）

問題5: (to / seems / gone / have / He) - 彼は行ってしまったようだ。

解答: He seems to have gone.

解説: 「to have 過去分詞」を使って完了形の不定詞を表現しています。

  • He: 彼（代名詞）
  • seems: ～のようだ（動詞）
  • to: ～すること（不定詞）
  • have: ～した（完了形）
  • gone: 行ってしまった（過去分詞）
