接続詞

文法と構文の説明

接続詞 は文と文、または文中の語句をつなぐ働きをします。 「because」「when」「if」は次のように使われます：

  • because： 理由を表します。 例：I stayed home because it was raining.（雨が降っていたので家にいました。）
  • when： 時を表します。 例：Call me when you arrive.（到着したら電話してください。） When you arrive, call me.
  • if： 条件を表します。 例：If it rains, we will stay home.（もし雨が降ったら、家にいます。） We will stay home If it rains.

問題

以下の単語を並び替えて正しい文を作成しましょう。（和訳を参考にしてください）

問題1: (stayed / it / home / because / raining / I / was) - 雨が降っていたので家にいました。

解答: I stayed home because it was raining.

解説: 「because」を用いて理由を表しています。

  • I: 私（主語）
  • stayed: 滞在した（動詞）
  • home: 家で（副詞）
  • because: ～なので（接続詞）
  • it: それ（主語）
  • was: ～だった（動詞）
  • raining: 雨が降っている（動名詞）

問題2: (arrive / me / you / when / call) - 到着したら電話してください。

解答: Call me when you arrive.

解説: 「when」を用いて時を表しています。

  • Call: 呼ぶ（動詞）
  • me: 私を（代名詞）
  • when: ～するとき（接続詞）
  • you: あなた（主語）
  • arrive: 到着する（動詞）

問題3: (it / we / will / stay / rains / if / home / ,) - もし雨が降ったら、家にいます。

解答: If it rains, we will stay home.

解説: 「if」を用いて条件を表しています。

  • If: もし～なら（接続詞）
  • it: それ（主語）
  • rains: 雨が降る（動詞）
  • we: 私たち（主語）
  • will: ～する予定（助動詞）
  • stay: 滞在する（動詞）
  • home: 家で（副詞）

問題4: (child / friends / I / soccer / when / I / with / played / my / was) - 子どもの頃、友達とサッカーをしました。

解答: I played soccer with my friends when I was a child.

解説: 「when」を使って時を述べています。

  • I: 私（主語）
  • played: ～した（動詞）
  • soccer: サッカー（名詞）
  • with my friends: 友達と（前置詞句）
  • when: ～とき（接続詞）
  • I : 私は（主語）
  • was: ～だった（動詞）
  • child: 子ども（名詞）

問題5: (he / late / was / because / missed / he / train / the) - 彼は電車に乗り遅れたので遅刻しました。

解答: He was late because he missed the train.

解説: 「because」を使って理由を述べています。

  • He: 彼（主語）
  • was: ～でった（動詞）
  • late: 遅れている（形容詞）
  • because: ～なので（接続詞）
  • missed: 逃した（動詞）
  • the: その（冠詞）
  • train: 電車（名詞）
