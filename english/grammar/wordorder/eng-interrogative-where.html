疑問詞 Where

文法と構文の説明

Where は「どこ」を尋ねる疑問詞です。 以下の3つの形式で使用されます：

  • Where + be動詞 - 主語の位置や存在を尋ねる。
  • Where + Do + S + V - 主語の動作が行われる場所を尋ねる。
  • Where + 助動詞 + S + V - 主語の未来や可能性の動作が行われる場所について尋ねる。

例文：

  • Where is the library? （図書館はどこですか？）
  • Where do you live? （あなたはどこに住んでいますか？）
  • Where will they meet? （彼らはどこで会う予定ですか？）

問題

以下の単語を並び替えて正しい文を作成しましょう。（和訳を参考にしてください）

問題1: (is / where / my / phone / ?) - 私の電話はどこですか？

解答: Where is my phone?

解説: be動詞「is」を用いて、物の場所を尋ねる疑問文を作成します。

  • Where: どこ（疑問詞）
  • is: ～である（be動詞）
  • my: 私の（代名詞）
  • phone: 電話（名詞）

問題2: (where / you / do / work / ?) - あなたはどこで働いていますか？

解答: Where do you work?

解説: 疑問詞「Where」に助動詞「do」を用い、働く場所を尋ねます。

  • Where: どこ（疑問詞）
  • do: ～する（助動詞）
  • you: あなた（代名詞）
  • work: 働く（動詞）

問題3: (where / they / meet / will / ?) - 彼らはどこで会いますか？

解答: Where will they meet?

解説: 助動詞「will」を用いて、未来の会う場所を尋ねます。

  • Where: どこ（疑問詞）
  • will: ～だろう（助動詞）
  • they: 彼ら（代名詞）
  • meet: 会う（動詞）

問題4: (the / where / nearest / station / is / ?) - 一番近い駅はどこですか？

解答: Where is the nearest station?

解説: be動詞「is」を用いて、一番近い駅の場所を尋ねます。

  • Where: どこ（疑問詞）
  • is: ～である（be動詞）
  • the: その（冠詞）
  • nearest: 最も近い（形容詞）
  • station: 駅（名詞）

問題5: (you / where / travel / will / next / ?) - 次にどこを旅行しますか？

解答: Where will you travel next?

解説: 助動詞「will」を用いて、次の旅行先を尋ねます。

  • Where: どこ（疑問詞）
  • will: ～だろう（助動詞）
  • you: あなた（代名詞）
  • travel: 旅行する（動詞）
  • next: 次に（副詞）
