be動詞 being

問題

以下の単語を正しい語順に並べ替えて、正しい英文を作りましょう。

問題 1: (a / being / is / doctor / he / .) - 彼は医者としてふるまっています。

解答: He is being a doctor.

問題 2: (is / being / kind / she / today / .) - 彼女は今日は優しくしています。

解答: She is being kind today.

問題 3: (being / careful / are / you / .) - あなたは注意深くしています。

解答: You are being careful.

問題 4: (being / quiet / is / the / class / .) - クラスは静かになっています。

解答: The class is being quiet.

問題 5: (is / being / the / room / cleaned / .) - その部屋は掃除されています。

解答: The room is being cleaned.

問題 6: (are / being / painted / walls / the / .) - 壁が塗られています。

解答: The walls are being painted.

問題 7: (was / being / friendly / he / very / .) - 彼はとても親しげにふるまっていました。

解答: He was being very friendly.

問題 8: (being / rude / stop / please / .) - 失礼な態度をやめてください。

解答: Please stop being rude.

問題 9: (is / becoming / being / it / cold / outside / .) - 外は寒くなっています。

解答: It is being cold outside.
