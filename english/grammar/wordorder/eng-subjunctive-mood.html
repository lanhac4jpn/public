仮定法

文法と構文の説明

仮定法は、現実と異なる仮定や願望を表す際に用いられます。以下のような表現があります：

  • 仮定法過去： If I were rich, I could travel the world.（もしお金持ちなら、世界中を旅行できるのに。）
  • 仮定法過去完了： If I had studied harder, I might have passed the exam.（もっと一生懸命勉強していたら、試験に合格していたかもしれない。）
  • wish： I wish I could fly.（飛べたらいいのに。）

主に助動詞（could, would, might）を用いて、現実では不可能なことや過去の後悔を表現します。

問題

以下の単語を並べ替えて正しい文を作成しましょう。（和訳を参考にしてください）

問題1: (were / If / I / rich / could / I / travel / the world / ,) - もしお金持ちなら、世界中を旅行できるのに。

解答: If I were rich, I could travel the world。

解説: 仮定法過去を用いて、現在の事実と異なる仮定を表現しています。

  • If: もし～なら（接続詞）
  • I: 私（主語）
  • were: ～であれば（動詞の仮定法過去形）
  • rich: お金持ちの（形容詞）
  • could: ～できる（助動詞）
  • travel: 旅行する（動詞）
  • the world: 世界（名詞句）

問題2: (wish / I / were / I / smarter) - 私がもっと賢かったらいいのに。

解答: I wish I were smarter。

解説: 仮定法過去を用いて、現在の願望を表現しています。

  • I: 私（主語）
  • wish: ～であればよいのにと思う（動詞）
  • were: ～であれば（動詞の仮定法過去形）
  • smarter: もっと賢い（形容詞）

問題3: (might / the exam / had / studied / I / if / harder / I / have / passed / ,) - もっと一生懸命勉強していたら、試験に合格していたかもしれない。

解答: If I had studied harder, I might have passed the exam。

解説: 仮定法過去完了を用いて、過去の事実と異なる仮定を表現しています。

  • If: もし～なら（接続詞）
  • I: 私（主語）
  • had studied: 勉強していた（過去完了形）
  • harder: もっと一生懸命に（副詞）
  • might: ～かもしれない（助動詞）
  • have passed: 合格した（現在完了形）
  • the exam: 試験（名詞句）

問題4: (any job / you / could / if / what / would / it / be / choose/ ?) - もしどんな仕事でも選べるなら、それは何ですか？

解答: If you could choose any job, what would it be?

解説: 仮定法過去を用いて、現在の仮定を表現しています。

  • If: もし～なら（接続詞）
  • you: あなた（主語）
  • could: ～できる（助動詞）
  • choose: 選ぶ（動詞）
  • any job: どんな仕事でも（名詞句）
  • what: 何（疑問詞）
  • would: ～だろう（助動詞）
  • it: それ（代名詞）
  • be: ～である（動詞）

解答: I wish I were taller。

  • wish: ～であればよいのにと思う（動詞）
  • were: ～であれば（動詞の仮定法過去形）
  • taller: もっと背が高い（形容詞）
