talk 「話す」

文法説明

「talk」は、誰かとやり取りするように「話す」「会話する」動作を表す動詞です[cite: 20]。話し合い、相談、雑談などでよく使われます[cite: 20]。

  • 現在形： talk／talks（三人称単数）[cite: 20]
  • 過去形： talked[cite: 20]
  • 進行形： is talking（〜しているところ）[cite: 20]
  • 命令形： Don’t talk.（話さないで）[cite: 20]

問題 1： (I / want / to / talk / to / you) - 私はあなたと話したいです[cite: 20]。

解答： I want to talk to you.[cite: 20]

解説： 「I want to talk to you.」は「私はあなたと話したいです。」という意味です[cite: 20]。

  • I： 私（代名詞）[cite: 20]
  • want： 〜したい（動詞）[cite: 20]
  • to： 〜すること（不定詞）[cite: 20]
  • talk： 話す（動詞）[cite: 20]
  • to： 〜に（前置詞）[cite: 20]
  • you： あなた（代名詞）[cite: 20]
