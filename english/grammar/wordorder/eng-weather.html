天候の表現

文法説明

天候を表す表現では「It is ～」がよく使われます。

  • 肯定文： It is sunny today.（今日は晴れです。）
  • 疑問文： Is it cold outside?（外は寒いですか？）
  • 否定文： It is not raining now.（今は雨が降っていません。）

短縮形

短縮形は会話でよく使われます。

  • It is → It's
  • It is not → It's not / It isn't

問題

以下の単語を並べ替えて正しい文章を作りましょう。（和訳も参考にしてください）

問題 1： (sunny / today / It / is) - 今日は晴れです。

解答： It is sunny today.

解説： 天候を表す「It is ～」の形を使います。

  • It： それ（代名詞、主格）
  • is： ～です（be動詞）
  • sunny： 晴れた（形容詞）
  • today： 今日（副詞）

問題 2： (Is / cold / outside / it / ?) - 外は寒いですか？

解答： Is it cold outside?

解説： 疑問文では「Is」を文頭に置きます。

  • Is： 〜ですか（be動詞、疑問文の先頭）
  • it： それ（代名詞、主格）
  • cold： 寒い（形容詞）
  • outside： 外で（副詞）

問題 3： (raining / not / is / It) - 雨が降っていません。

解答： It is not raining.

短縮形： It's not raining. または It isn't raining.

解説： 否定文では「is」の後に「not」を置きます。

  • It： それ（代名詞、主格）
  • is： 〜です（be動詞）
  • not： 〜でない（否定詞）
  • raining： 雨が降る（動詞）

問題 4： (It / cloudy / is / very) - とても曇っています。

解答： It is very cloudy.

解説： 「very」を加えて程度を表しています。

  • It： それ（代名詞、主格）
  • is： 〜です（be動詞）
  • very： とても（副詞）
  • cloudy： 曇った（形容詞）

問題 5： (will / sunny / tomorrow / It / be) - 明日は晴れるでしょう。

解答： It will be sunny tomorrow.

解説： 未来を表す「will」を使っています。

  • It： それ（代名詞、主格）
  • will： 〜でしょう（助動詞）
  • be： 〜である（動詞の原形）
  • sunny： 晴れた（形容詞）
  • tomorrow： 明日（副詞）
