助動詞 dare

文法と構文の説明

助動詞 dare は、「あえて～する」「大胆に～する」という意味で、挑戦や恐れを伴う行動を表します。

  • 肯定文: 主語 + dare + 動詞の原形
  • 疑問文: Dare + 主語 + 動詞の原形?
  • 否定文: 主語 + dare not + 動詞の原形

例文：

  • 肯定文: He dare speak the truth. （彼はあえて真実を話す。）
  • 疑問文: Dare she try this challenge? （彼女はこの挑戦を試みるだろうか？）
  • 否定文: I dare not disobey my parents. （私は両親に逆らう勇気がない。）

問題

以下の単語を並び替えて正しい文を作成しましょう。（和訳を参考にしてください）

問題1: (truth / speak / dare / he / the) - 彼はあえて真実を話す。

解答: He dare speak the truth.

解説: 肯定文では「主語 + dare + 動詞の原形」を使います。

  • He: 彼（主語）
  • dare: あえて～する（助動詞）
  • speak: 話す（動詞の原形）
  • the: その（冠詞）
  • truth: 真実（名詞）

問題2: (she / try / dare / this / challenge / ?) - 彼女はこの挑戦を試みるだろうか？

解答: Dare she try this challenge?

解説: 疑問文では「Dare + 主語 + 動詞の原形」を使います。

  • Dare: あえて～する（助動詞）
  • she: 彼女（主語）
  • try: 試みる（動詞の原形）
  • this: この（指示詞）
  • challenge: 挑戦（名詞）

問題3: (not / I / dare / disobey / my / parents) - 私は両親に逆らう勇気がない。

解答: I dare not disobey my parents.

解説: 否定文では「主語 + dare not + 動詞の原形」を使います。

  • I: 私（主語）
  • dare: あえて～する（助動詞）
  • not: ～ない（否定語）
  • disobey: 逆らう（動詞の原形）
  • my: 私の（所有格）
  • parents: 両親（名詞）

問題4: (face / they / dare / the / risk) - 彼らはあえてリスクに立ち向かう。

解答: They dare face the risk.

解説: 肯定文では「主語 + dare + 動詞の原形」を使います。

  • They: 彼ら（主語）
  • dare: あえて～する（助動詞）
  • face: 立ち向かう（動詞の原形）
  • the: その（冠詞）
  • risk: リスク（名詞）

問題5: (dare / not / he / reveal / secret / the) - 彼はあえて秘密を明かさない。

解答: He dare not reveal the secret.

解説: 否定文の形式を使います。

  • He: 彼（主語）
  • dare: あえて～する（助動詞）
  • not: ～ない（否定語）
  • reveal: 明かす（動詞の原形）
  • the: その（冠詞）
  • secret: 秘密（名詞）
