関係代名詞 非制限用法 which

文法説明

関係代名詞の非制限用法「which」は、主文の内容に追加情報を加えるために使われます[cite: 20]。この「which節」は主文とカンマで区切られ、主文全体またはその一部を補足する役割を果たします[cite: 20]。

例：

  • 肯定文： The Earth revolves around the Sun, which takes about 365 days. （地球は太陽の周りを回っており、それには約365日かかります）[cite: 20]

用語解説

関係代名詞「which」の非制限用法に関連する用語：

  • 関係代名詞 (Relative Pronoun)： 主文の情報に付加的な情報を追加します[cite: 20]。
  • 非制限用法 (Non-defining Clause)： 主文の内容に限定を加えず、追加情報として機能する節です[cite: 20]。

問題

以下の単語を並べ替えて正しい文章を作りましょう。（和訳も参考にしてください）[cite: 20]

問題 1： (the Sun / revolves / Earth / the / which / takes / around / about 365 days) - 地球は太陽の周りを回っており、それには約365日かかります[cite: 20]。

解答： The Earth revolves around the Sun, which takes about 365 days.[cite: 20]

解説： 「which」は主文の「太陽の周りを回ること」に対して「それには365日かかる」という補足情報を追加しています[cite: 20]。

  • The： ～は（定冠詞）[cite: 20]
  • Earth： 地球（名詞）[cite: 20]
  • revolves： 回る（動詞）[cite: 20]
  • around： ～の周りを（前置詞）[cite: 20]
  • the Sun： 太陽（名詞句）[cite: 20]
  • which： それは（関係代名詞）[cite: 20]
  • takes： かかる（動詞）[cite: 20]
  • about： 約（副詞）[cite: 20]
  • 365 days： 365日（名詞句）[cite: 20]

問題 2： (this book / written / is / by my friend / which / has / won several awards) - この本は私の友人が書いたもので、いくつかの賞を受賞しています[cite: 20]。

解答： This book is written by my friend, which has won several awards.[cite: 20]

解説： 「which」は「本」に対して「いくつかの賞を受賞している」という補足情報を追加しています[cite: 20]。

  • This： この（指示形容詞）[cite: 20]
  • book： 本（名詞）[cite: 20]
  • is written： 書かれている（動詞）[cite: 20]
  • by： ～によって（前置詞）[cite: 20]
  • my friend： 私の友人（名詞句）[cite: 20]
  • which： それは（関係代名詞）[cite: 20]
  • has： 持っている（動詞）[cite: 20]
  • won： 勝ち取った（動詞）[cite: 20]
  • several awards： いくつかの賞（名詞句）[cite: 20]

解説： 「which」が「車」に対する補足情報として「それは高価である」という意味を加えています[cite: 20]。

  • father： 父（名詞）[cite: 20]
