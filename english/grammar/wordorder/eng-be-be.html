be動詞 be

文法説明

「be動詞の原形」は、不定詞（to be）、命令文、または仮定法などで使われます。 主語に関係なく使用される点が特徴です。

  • 肯定文: I want to be a doctor.（私は医者になりたい。）
  • 疑問文: Can it be true?（それは本当であり得ますか？）
  • 否定文: It must not be difficult.（それは難しいはずがない。）

問題

以下の単語を並び替えて正しい文章を作りましょう。（和訳も参考にしてください）

問題1: (want / to / be / I / happy) - 私は幸せでありたい。

解答: I want to be happy。

解説: 「want to be」で「～でありたい」という意味を表します。

  • I: 私（代名詞、主格）
  • want: ～したい（動詞）
  • to: ～することを（不定詞）
  • be: ～である（動詞の原形）
  • happy: 幸せな（形容詞）

問題2: (true / it / can / be / ?) - それは本当であり得ますか？

解答: Can it be true?

解説: 疑問文では助動詞「Can」を文頭に置きます。

  • Can: ～できますか（助動詞）
  • it: それ（代名詞、主格）
  • be: ～である（動詞の原形）
  • true: 本当の（形容詞）

問題3: (must / easy / not / be / It) - それは簡単なはずがない。

解答: It must not be easy。

解説: 「must not」で「～してはならない」「～であるはずがない」を表します。

  • It: それは（代名詞、主格）
  • must: ～しなければならない（助動詞）
  • not: ～でない（否定詞）
  • be: ～である（動詞の原形）
  • easy: 簡単な（形容詞）

問題4: (to / or / to / not / be / be) - 生きるべきか、死ぬべきか。

解答: To be or not to be。

解説: シェイクスピアの有名な一節。 「to be」で存在を意味します。

  • to: ～することを（不定詞）
  • be: ～である（動詞の原形）
  • or: または（接続詞）
  • not: ～でない（否定詞）
  • to: ～することを（不定詞）
  • be: ～である（動詞の原形）

問題5: (should / be / careful / They) - 彼らは注意すべきです。

解答: They should be careful。

解説: 助動詞「should」は「～すべき」を意味します。

  • They: 彼らは（代名詞、主格）
  • should: ～すべき（助動詞）
  • be: ～である（動詞の原形）
  • careful: 注意深い（形容詞）
