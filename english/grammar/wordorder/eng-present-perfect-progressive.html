現在完了進行形

文法と構文の説明

現在完了進行形は、過去に始まった動作が現在も続いていることを表します。

現在完了進行形の構造は次の通りです：

  • 肯定文: 主語 + have/has + been + 現在分詞
  • 疑問文: Have/Has + 主語 + been + 現在分詞 + ...?
  • 否定文: 主語 + have/has + not + been + 現在分詞

例文：

  • He has been studying English for two hours.（彼は2時間英語を勉強しています。）
  • Have you been waiting here long?（あなたはここで長い間待っていますか？）
  • I have not been feeling well lately.（私は最近体調が良くありません。）

問題

以下の単語を並び替えて正しい文を作成しましょう。（和訳を参考にしてください）

問題1: (been / has / He / for / studying / hours / English / two) - 彼は2時間英語を勉強しています。

解答: He has been studying English for two hours。

解説: 現在完了進行形で「have/has + been + 現在分詞」を使用しています。

  • He: 彼（主語）
  • has: ～している（助動詞）
  • been: ～している（助動詞の一部）
  • studying: 勉強している（現在分詞）
  • English: 英語（名詞）
  • for: ～の間（前置詞）
  • two hours: 2時間（時間表現）

問題2: (you / waiting / Have / long / here / been/ ?) - あなたはここで長い間待っていますか？

解答: Have you been waiting here long?

解説: 疑問文の形で「Have + 主語 + been + 現在分詞」を使用しています。

  • Have: ～していますか（助動詞）
  • you: あなた（主語）
  • been: ～している（助動詞の一部）
  • waiting: 待っている（現在分詞）
  • here: ここで（副詞）
  • long: 長い間（副詞）

問題3: (lately / not / have / been / well / feeling / I) - 私は最近体調が良くありません。

解答: I have not been feeling well lately。

解説: 否定文の形で「have not + been + 現在分詞」を使用しています。

  • I: 私（主語）
  • have: ～している（助動詞）
  • not: ～していない（否定語）
  • been: ～している（助動詞の一部）
  • feeling: 感じている（現在分詞）
  • well: 良い（形容詞）
  • lately: 最近（副詞）

問題4: (they / working / been / not / have / on / project / this) - 彼らはこのプロジェクトに取り組んでいません。

解答: They have not been working on this project。

解説: 否定文の形で現在完了進行形を使用しています。

  • They: 彼ら（主語）
  • have: ～している（助動詞）
  • not: ～していない（否定語）
  • been: ～している（助動詞の一部）
  • working: 働いている（現在分詞）
  • on: ～に関して（前置詞）
  • this project: このプロジェクト（名詞句）

問題5: (have / you / been / studying / English / hard / recently?) - あなたは最近一生懸命英語を勉強していますか？

解答: Have you been studying English hard recently?

解説: 疑問文の形で「Have + 主語 + been + 現在分詞」を使用しています。

  • Have: ～していますか（助動詞）
  • you: あなた（主語）
  • been: ～している（助動詞の一部）
  • studying: 勉強している（現在分詞）
  • English: 英語（名詞）
  • hard: 一生懸命に（副詞）
  • recently: 最近（副詞）
