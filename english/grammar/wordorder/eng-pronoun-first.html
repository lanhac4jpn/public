代名詞 一人称と二人称

文法説明

代名詞には話し手自身や話し手を含むグループを指す「一人称」と、話し手が相手に直接言及する「二人称」があります。

  • 一人称: I（私）、We（私たち）
  • 二人称: You（あなた、あなたたち）

短縮形

主格の代名詞「I」「We」「You」がbe動詞と組み合わさる場合、短縮形を使うことが一般的です。

  • 例:
  • I am → I'm
  • We are → We're
  • You are → You're

用語解説

  • 主格 (Subjective Case): 主語として使われる代名詞。
  • 一人称 (First Person): 話し手自身または話し手を含むグループを指します。
  • 二人称 (Second Person): 話し手が相手に直接言及するときに使います。

問題

以下の単語を並べ替えて正しい文章を作りましょう。（和訳も参考にしてください）

問題1: (books / like / reading / I) - 私は本を読むのが好きです。

解答: I like reading books.

解説: 「I」が主格として主語の位置にあります。

  • I: 私は（代名詞、主格）
  • like: 好き（動詞）
  • reading: 読むこと（動名詞）
  • books: 本（名詞、複数形）

問題2: (busy / are / very / we / today) - 私たちは今日とても忙しいです。

解答: We are very busy today.

短縮形: We're very busy today.

解説: 「We」が主格として主語の位置にあります。

  • We: 私たちは（代名詞、主格）
  • are: ～です（be動詞）
  • very: とても（副詞）
  • busy: 忙しい（形容詞）
  • today: 今日（副詞）

問題3: (everyone / are / kind / you / to ) - あなたはみんなに親切です。

解答: You are kind to everyone.

短縮形: You're kind to everyone.

解説: 「You」が主格として主語の位置にあります。

  • You: あなたは（代名詞、主格）
  • are: ～です（be動詞）
  • kind: 親切（形容詞）
  • to: ～に対して（前置詞）
  • everyone: みんな（代名詞）

問題4: (soccer / enjoy / we / playing / together) - 私たちは一緒にサッカーをするのを楽しみます。

解答: We enjoy playing soccer together.

解説: 「We」が主格として使われています。

  • We: 私たちは（代名詞、主格）
  • enjoy: 楽しむ（動詞）
  • playing: ～すること（動名詞）
  • soccer: サッカー（名詞）
  • together: 一緒に（副詞）

問題5: (you / I / are / honest / think) - 私はあなたが正直だと思います。

解答: I think you are honest.

解説: 「I」が主格として主語の位置にあり、「you」は補語です。

  • I: 私は（代名詞、主格）
  • think: 思う（動詞）
  • you: あなたは（代名詞、主格）
  • are: ～です（be動詞）
  • honest: 正直（形容詞）
