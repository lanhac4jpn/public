be動詞 is

文法説明

be動詞「is」は、主語が三人称単数（he, she, it）または単数名詞の場合に使われます。

  • 肯定文: She is a doctor.（彼女は医者です。）
  • 疑問文: Is he a student?（彼は学生ですか？）
  • 否定文: It is not easy.（それは簡単ではありません。）

短縮形

短縮形は主に会話で使われます。

  • He is → He's
  • She is → She's
  • It is → It's
  • It is not → It's not / It isn't

問題

以下の単語を並べ替えて正しい文章を作りましょう。（和訳も参考にしてください）

問題1: (is / teacher / a / she) - 彼女は教師です。

解答: She is a teacher.

解説: 主語「She」に対してbe動詞「is」を使い、職業を表します。

  • She: 彼女は（代名詞、主格）
  • is: ～です（be動詞）
  • a: 一つの（冠詞）
  • teacher: 教師（名詞）

問題2: (is / happy / he / ?) - 彼は幸せですか？

解答: Is he happy?

解説: 疑問文ではbe動詞「is」を文頭に置きます。

  • Is: ～ですか（be動詞、疑問文の先頭）
  • he: 彼は（代名詞、主格）
  • happy: 幸せな（形容詞）

問題3: ( is / difficult / it / not / ?) - それは難しくありません。

解答: It is not difficult.

短縮形: It's not difficult. または It isn't difficult.

解説: 否定文では「is」の後に「not」を置きます。

  • It: それは（代名詞、主格）
  • is: ～です（be動詞）
  • not: ～でない（否定詞）
  • difficult: 難しい（形容詞）

問題4: (is / dog / outside / the) - その犬は外にいます。

解答: The dog is outside.

解説: 主語「The dog」に対してbe動詞「is」を使い、場所を表します。

  • The: その（冠詞）
  • dog: 犬（名詞）
  • is: ～です（be動詞）
  • outside: 外に（副詞）

問題5: (table / it / on / is / the / ?) - それはテーブルの上にありますか？

解答: Is it on the table?

解説: 疑問文の形を保ち、「is」を先頭に置きます。

  • Is: ～ですか（be動詞、疑問文の先頭）
  • it: それは（代名詞、主格）
  • on: ～の上に（前置詞）
  • the: その（冠詞）
  • table: テーブル（名詞）
